# ⚙️ LG SmartCare — Predictive Maintenance AI

A machine learning web application that predicts appliance failure risk based on sensor data. Built to simulate how a real-world IoT predictive maintenance system works — inspired by LG's SmartCare platform.

---

## 🎯 Problem Statement

Industrial appliances and home machines fail unexpectedly, causing costly repairs and downtime. This project uses historical sensor data to **predict failure before it happens**, allowing timely preventive maintenance.

> Reactive repair cost: ₹5,000 | Preventive maintenance cost: ₹1,500 → **Saves ₹3,500 per machine**

## 🔄 Project Phases

### Phase 1 — Predictive Maintenance (ML Pipeline)
- Loaded and cleaned the **AI4I 2020 Predictive Maintenance** dataset (10,000 records)
- Renamed columns to match LG SmartCare schema
- Feature engineering: `temp_diff` (process − ambient temperature), `device_age_months`
- Dropped sub-failure columns (TWF, HDF, PWF, OSF, RNF) to **prevent data leakage**
- Handled severe class imbalance using **SMOTE** (only on training data)
- Trained **XGBoost classifier** with custom threshold (0.35) optimized for recall
- Exported model and preprocessor using `joblib`

### Phase 2 — Energy Analysis
- Loaded appliance energy dataset (`ape.csv`)
- Extracted time-based features: hour, day of week, weekend flag
- Aggregated daily energy consumption (kWh), peak usage hours
- Simulated regional business context (North/South/East/West) for demonstration

### Phase 3 — Business KPIs
- Calculated failure rate, high-risk device count, average risk score
- Estimated **cost savings** from preventive vs reactive maintenance
- Merged ML predictions with energy data for a unified device view
